# RAG-based-ai-model
RAG-Based AI Teaching Assistant

This project is an intelligent AI-powered Teaching Assistant built using a Retrieval-Augmented Generation (RAG) architecture. It allows users to upload PDF documents , videos(MP4) and ask questions, and the system generates accurate, context-aware answers based on the uploaded content.

The application combines Large language model with modern AI techniques to enhance information retrieval and provide human-like responses.

🚀 Features:
- 📄 Upload and process PDF documents and videos mp4 format
- 🔍 Context-based question answering
- 🧠 Uses RAG (Retrieval-Augmented Generation) pipeline
- 💬 Interactive chat interface using Streamlit
- ⚡ Fast and efficient response generation

🛠️ Tech Stack:
- Python
- Streamlit (Frontend UI)
- LangChain / Custom RAG pipeline
- Vector Database (FAISS / Chroma)
- OpenAI / LLM models

📌 Use Cases:
- AI-powered learning assistant
- Document-based Q&A system
- Knowledge retrieval from custom data

  steps:
- read data - convert to json - chunk - embeddings

🎯 Objective:
To build an AI system that can understand user queries and provide precise answers by retrieving relevant information from documents, instead of relying only on pre-trained knowledge.
